selectores simples

  1. Selector de etiqueta:Para cambiar la forma general de una etiqueta general, por ejemplo a{} - a siempre genera una linea debajo del texto, si quisiera removerlo en todo el trabajo, se podría quitar.
  2. Selector de clase: Para realizar estilos en la web, en la mayoria de casos se recomienda selector de clase, ya que es el valor más bajo que te permite especificidad, en casos especiales se podría usar otro.

    3. .logo{ text-align: center; }

  3. Selector de ID: No se recomienda usar para CSS, se prefiere que se usen para la programación con JAVASCRIPT.

    4. #logo{ text-align:center; }

  4. Selector InLine: Solo se debe usar para JavaScript para aplicar estilos. En Javascript se aplica estilos InLine.

selectores compuestos

  1. Selector descendiente: se utilizan para especificar selectores dentro de otro

    2. header h1{ font-size: 3rem; text-align: center; background-color: #FDFFFC ; }

  2. Selector hijo directo: solo afectaría al hijo directo del selector inicial

    3. h2 >a{ text-decoration: none; background-color: #FDFFFC; color: black; }

  3. Selector de hermano siguiente o adyacente: +, selecciona a la etiqueta que esta al mismo nivel y en la siguiente linea del hermano

    4. h2 + p{ color:red}

  4. Selector a hermanos siguientes o adyacentes: ~, aplica a todos los hermanos después (no necesita ser inmediato despues), no aplica a nietos.

    5. h2 ~p{ color:red}

  5. Selector multiples etiquetas: seleccionar varios elementos a la vez

    6. h2,p{ color:red}

box model

    Partes del box model:

  1. Box content: parte principal que contiene el texto o valores del elemento
  2. Box padding: espacio entre el contenido y el borde
  3. Box border: limites de la caja
  4. Box margin: parte externa, seria el espacio entre los elementos

    Aplicacion de atributos:

  1. Aplicando box sizing: esto vuelve que el width y height sean absolutos, el border, padding y contenido van dentro

    2. *{box-sizing: border-box} aplicado a este cuadrado

  2. Aplicando padding y margin

    3. .caja{padding 20px 10px 5px 3px; margin 20px 10px;}

  3. Colapsando margenes verticales

    4. .caja2{margin-top=2rem} .caja1{margin-bottom=4rem}

  4. Margenes negativos

    5. .caja2{ margin-top: 3rem; margin-bottom: -1rem; } .caja3{ margin-left: -2rem; margin-bottom: 1rem; background-color: rgb(6, 8, 10); }

  5. Centrado

    6. .caja4{ margin: 0 auto; }
